# Quản lý bán trà sữa

## Mô tả
Phần mềm quản lý bán trà sữa được xây dựng bằng WinForms, phục vụ cho đồ án cơ sở 01.

## Chức năng - Admin
- Quản lý tài khoản
- Quản lý sản phẩm
- Quản lý nhân viên
- Quản lý khách hàng
- Quản lý chương trình khuyến mãi
- Quản lý đơn hàng
- Quản lý nhật ký hoạt động của hệ thống

## Chức năng - Nhân viên
- Quản lý thông tin
- Quản lý khách hàng
- Quản lý sản phẩm
- Bán hàng
- Quản lý hóa đơn đã lập
## 🛠 Công nghệ sử dụng
- C#
- .NET Framework (WinForms)
- SQL Server
- ADO.NET
